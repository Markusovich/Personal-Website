Personal Website



ABOUT

I created this website to showcase my achievements and demonstrate my ability in website development.

HTML & CSS

HTML & CSS features include but are not limited to: title, meta, style, div, p, hr, br, blockquote, q, cite, ul, li, header, footer, nav, section, article, aside, figure, figcaption, a, class selectors, id selectors, color properties, text properties, box-shadow, box model, sticky header, and parallax scrolling.

I have used HTML & CSS in a way to make this website in a creative fashion and to maximize it's neatness and style.

JAVASCRIPT

Website contains a dash of javascript.

Used javascript to create dynamic onclick functions, that do a variety of actions.

Actions include; form validation, and pop-up confirmations.

NODE & EXPRESS

Nodejs and Expressjs used as server side work.

Using this software has enabled me to store form data, use form data to send emails to recipient, and host website online.

MONGODB

Mongodb and it's database technology has organized my data greatly.

I am now able to store user feedback data in my own database.

Mongodb has allowed me to use database information, such as the size of the database, or if a certain user is already in the database.

ANGULAR

I have not used many angular features or technology, but I have enabled output value live on ng-click.
